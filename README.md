# 🎮 Tic Tac Toe Ultimate

A simple yet interactive **2-player Tic Tac Toe** game developed in **C++**. This console-based application provides an enjoyable gaming experience with player name customization, score tracking, file handling, and sound effects.

---

## 📌 Features

- 🎯 Two-player gameplay (Player X vs Player O)
- 👤 Custom player names
- 📊 Live scoreboard
- 💾 Save and load scores using file handling
- 🔄 Reset scoreboard option
- ✅ Input validation for valid moves
- 🏆 Automatic win and draw detection
- 🔊 Sound effects (Windows)
- 🖥️ User-friendly menu-driven interface

---

## 🛠️ Technologies Used

- C++
- Object-Oriented Programming (OOP)
- File Handling (`ifstream` & `ofstream`)

## 🎮 How to Play

- Select **New Game** or **Quick Play**.
- Enter player names (optional).
- Players take turns entering **Row** and **Column** numbers.
- First player uses **X**, second player uses **O**.
- The first player to align **three symbols** horizontally, vertically, or diagonally wins.
- If all cells are filled without a winner, the match ends in a draw.
